Built for the database work you actually do

Connect and start working without installing or configuring IBM ODBC drivers. JustyBase includes its own Node.js-based Netezza driver and keeps the familiar VS Code workflow close at hand.

  • Explore schemas, tables, views, procedures, and dependencies.
  • Write, validate, lint, and run Netezza SQL in one workspace.
  • Use Copilot with database context for explain, fix, optimize, and generate workflows.
  • Inspect query results, import/export data, and work with operational tooling.

Install manually (.vsix)

Use this when you need to install a release package directly from GitHub.

Download VSIX · v3.16.8
  1. Download the .vsix file for the core Netezza extension.
  2. In VS Code, open the Command Palette with Ctrl+Shift+P.
  3. Choose Extensions: Install from VSIX....
  4. Select the downloaded file and restart or reload VS Code if prompted.
Manual installation does not receive automatic updates through the Marketplace. Download a newer VSIX when you want to upgrade.
